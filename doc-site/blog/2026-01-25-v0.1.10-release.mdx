We are excited to announce the release of **v0.1.10** of the RDS Slint UI Kit! This update brings several new components and significant enhancements to existing ones, further expanding the toolkit for audio and creative applications.

{/* truncate */}

## What's New in v0.1.10

### New Components
- **RDSButton**: A standard, highly customizable button component.
- **RDSXYPad**: A 2D pad perfect for controlling dual parameters like X/Y filter cutoff and resonance.
- **RDSNumericKeypad**: A full numeric keypad designed for precise value entry on touch devices.
- **RDSKeybed**: A virtual piano keyboard for MIDI interaction and visualization.

### Enhanced Components
- **RDSKnob**: Now features a comprehensive labeling system with customizable positioning and styling.
- **RDSSelectButton**: Added `shift-clicked` support and configurable dimensions.
- **RDSToggle**: New styling properties for background, border, and icons.
- **RDSNumStepper**: Added `allow-editing` toggle and double-click callbacks for manual entry.
- **RDSVUMeter / RDSVertVUMeter**: Added `vu-border-width` property for finer visual control.
- **RDSSlider**: Improved internal structure and documentation.

### Documentation & Distribution
- Rewrote internal `.slint` documentation comments for better clarity.
- Fully updated the `README.md` with all new components and property details.
- Optimized asset inclusion (fonts, SVGs) in the release packages.

Check out the [Documentation](/docs/) to get started with these new features!
